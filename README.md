# 🎮 Pro Calc & Game Hub

A fully functional web-based calculator with a hidden "Stealth Mode" game console.

## 🚀 Features
* **Dual Theme:** Switch between Light/Professional and Dark/Code modes.
* **Secret Hub:** Access 100+ games via a secret code.
* **Panic Key:** Press `Esc` to instantly hide the games and return to the calculator.
* **Infinite Scroll:** Smooth browsing through a massive library of games.
* **JSON Powered:** Game data is stored in a clean `games.json` file for easy updates.

## 🕹️ How to Use
1. Use the calculator for standard math.
2. Enter the secret code `1234` and press `=`.
3. Browse, search, or filter games by category.
4. **Emergency?** Hit the `Esc` key to go back to "Math Mode" instantly.

## 🛠️ Tech Stack
* HTML5 / CSS3
* JavaScript (ES6+)
* JSON for data storage
